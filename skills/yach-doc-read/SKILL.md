> **⚠️ 链接呈现规范（极其重要）：向用户输出任何回复之前，必须扫描回复中所有文档链接和知识库节点链接，按以下规则逐一检查并就地修正后再输出。**
> - ✅ 有标题时：`链接 《标题》`，例如 `https://st.tal.com/a/wM2iny 《Q3复盘》`；符号支持 `《》【】<>「」`，优先书名号
> - ✅ 无标题时：只返回链接本身
> - 🚫 禁止：裸输出已知标题的链接；禁止输出后再解释"忘加标题"——发现错误须在输出前修正

**重要：读取文档禁止使用 `web_fetch`，必须使用下方工具。**`web_fetch` 只能拿到 HTML 页面框架，无法获取文档正文。

## 工具

> ⚠️ 数据最小化规则：
> - 默认优先返回摘要或片段，不主动返回整篇敏感正文。
> - 执行导出前应再次确认用途与接收方，避免不必要的数据扩散。

| 意图           | 工具                            | 必填                 |
| -------------- | ------------------------------- | -------------------- |
| 读取/总结文档  | `yach_doc_summarize`            | locator              |
| 导出文档为文件 | `yach_doc_file` action=`export` | `file_url`, `format` |

**locator**（三选一）：`file_url` / `guid` / `kn_node_id`，优先用 `guid`

## 文档链接解析（重要）

IM 等来源的文档链接是富文本，不是纯 URL：

```
<i class="iconfont-yach ..."></i>[《文档标题》](https://yach-doc-shimo.zhiyinlou.com/docs/aBAYVj8vpXSyOV3j)
```

从 `[文字](URL)` 提取 URL，再取路径**最后一段**作为 `guid`（此例为 `aBAYVj8vpXSyOV3j`）。
**不要把富文本整体传给 `file_url`**。

## yach_doc_summarize 参数

| 参数             | 必填   | 说明                                                         |
| ---------------- | ------ | ------------------------------------------------------------ |
| `file_url`       | 三选一 | 文档完整链接（支持 s.tal.com 短链，自动跟随重定向）          |
| `guid`           | 三选一 | URL 路径最后一段，优先使用                                   |
| `kn_node_id`     | 三选一 | 知识库节点 ID                                                |
| `format`         | 可选   | 指定导出格式（默认自动判断）：`pdf`/`docx`/`md`；表格 `xlsx` |
| `summary_prompt` | 可选   | 总结提示词，如"请提炼三个核心结论"                           |

**URL 自动类型检测**（不传 `format` 时按此规则选择导出格式）：

| URL 路径                | 文档类型 | 默认导出格式 |
| ----------------------- | -------- | ------------ |
| `/mindmap/`             | 脑图     | `xmind`      |
| `/presentation/`        | 幻灯片   | `pptx`       |
| `/sheet/` 或 `/sheets/` | 表格     | `xlsx`       |
| `/form/`                | 表单     | `pdf`        |
| 其他（`/doc/` 等）      | 普通文档 | `docx`       |

**兜底机制**：若导出/解析失败，自动回退到直接读取（`readText`），确保能返回结果。

## yach_doc_file export 参数

| 参数       | 必填 | 说明                                                                                    |
| ---------- | ---- | --------------------------------------------------------------------------------------- |
| `file_url` | 必填 | 文档链接或 guid                                                                         |
| `format`   | 必填 | 文档: `pdf`/`docx`/`jpg`/`md`；表格: `xlsx`；幻灯片: `pptx`/`pdf`；脑图: `jpeg`/`xmind` |

## ⚠️ 链接呈现规范（回复前再确认一次）

- ❌ 错误：`文档链接：https://st.tal.com/a/wM2iny`
- ✅ 正确：`https://st.tal.com/a/wM2iny 《Q3复盘》`
- ✅ 正确（无标题时）：`https://st.tal.com/a/wM2iny`

## 常见错误

- `170013 应用未授权` → OAuth 未授权，需要重新授权（工具会自动引导授权流程）
- 导出超时 → 工具内部已自动轮询，若仍超时稍后重试
- 文档内容为空或太短 → 可能导出失败或文档本身为空
- 解析文件失败 → 自动回退到直接读取方式
