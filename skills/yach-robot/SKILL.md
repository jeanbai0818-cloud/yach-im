## 工具：`yach_robot_groups`（使用 App Token，无需 OAuth 授权）

| 意图 | 工具名 | 必填参数 |
| --- | --- | --- |
| 查机器人所在群列表 | `yach_robot_groups` | 无 |

此工具无需任何参数，直接调用即可。

## 返回数据结构

```json
{
  "groups": [
    {
      "groupId": "2771361680",
      "name": "项目讨论群"
    }
  ]
}
```

| 字段 | 说明 |
| --- | --- |
| `groupId` | 群 ID，可用于后续发送消息 |
| `name` | 群名称 |

## 使用场景

### 向某个群发消息但不知道群 ID

先调用 `yach_robot_groups` 列出机器人所在的所有群，从返回列表中找到目标群的 `groupId`，再调用消息发送工具。

```
帮我查一下机器人在哪些群里
```

```
机器人在哪个群？我想给"技术交流群"发一条消息
```
